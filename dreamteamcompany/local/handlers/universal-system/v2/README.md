# v2 BP Project

Отдельный подпроект для архитектуры обработки лидов через Bitrix24 BizProc.

## Состав

- `generate_bp775.py` — генератор `bp775_lead_processor.bpt`
- `helpers/` — PHP-хелперы для локальной отладки и переиспользования логики BP
- `snippets/` — готовые тела для блоков `CodeActivity`
- `docs/` — схемы и рабочие заметки по BP-пайплайну

## Связанные файлы вне папки

- `universal/webhook_bp.php` — внешняя точка входа v2

## Назначение

Папка собирает все артефакты, относящиеся к варианту `v2`, чтобы BP-логика не смешивалась с корневым legacy-кодом и версиями `v3`/`v4`.
