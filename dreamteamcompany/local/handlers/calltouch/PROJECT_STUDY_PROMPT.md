# Промт для изучения проекта `calltouch`

Ты работаешь с проектом по пути `/project/dreamteamcompany/local/handlers/calltouch`.

Твоя задача: подробно изучить проект и создать отдельный файл с техническим описанием системы, чтобы по нему можно было безопасно вносить исправления, устранять баги и планировать улучшения.

## Что это за проект

По текущей структуре это PHP-проект интеграции `Calltouch` с `Bitrix24`, где:

- входящие данные по звонкам попадают в очередь JSON-файлов;
- основной обработчик читает очередь и создает или обновляет лиды в Bitrix24 через нативный API;
- часть логики завязана на инфоблоки Bitrix;
- есть веб-интерфейсы для управления очередью и конфигурацией;
- есть журналы, очередь ошибок и чат-уведомления.
